あなたのお金、正確にトラッキングできていますか？

Zanは、プロの会計士が使う複式簿記（Double-Entry Bookkeeping）の仕組みを、誰でも簡単に使えるようにしたスマート家計簿アプリです。

複式簿記と聞くと難しそうに感じますか？ご安心ください。Zanでは複式簿記が完全に見えない形で動作します。

あなたがすることは、「どこから → どこへ」を選ぶだけです。

例えば：
• コンビニで現金支払い → 「現金 → 食費」を選ぶだけ
• 給与振込 → 「給与 → 銀行口座」を選ぶだけ
• カード引き落とし → 「銀行口座 → クレジットカード」を選ぶだけ

このシンプルな入力だけで、Zanが裏側で借方と貸方を自動的に処理します。その結果、1円の誤差もない完璧な資産状況をお見せします。
